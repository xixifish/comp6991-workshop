# COMP6991 Workshop Questions

Weekly lab and workshop exercises for COMP6991 (Rust) at UNSW.

## Structure 

This is a Cargo workshop. Each week is a separate package.

- `week01/` -- Image Creation 
- `week02/` -- Rust Train-ing

## Running

From the workspace root:

```
cargo run -p week01 -- <path> <operation>
```
